# IIE Useful Utils - 基于 AstrBot 的群机器人的功能拓展

> 请注意：本插件并未正是在AstrBot市场中发布！本插件仅用于本地部署，为考研抄底群提供一些方便的功能。

## 💬历次更新

### 1.0 (Beta)
*   ✨更新了根据群聊中发言频率来限制机器人回复次数的功能，以保障群中没有人能够一直刷屏。


## 🧩现有工具一览
